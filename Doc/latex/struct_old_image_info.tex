\doxysubsection{Detailed Description}
This is an old version that should not be used anymore in favor of \doxylink{struct_image_info}{Image\+Info}. 

This type is still used in the critical curve finding and handling routines. The results of these routines should be returned in the form of and array of \doxylink{struct_image_info}{Image\+Info}\textquotesingle{}s now.
