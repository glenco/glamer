\doxysubsection{Detailed Description}
A class for calculating the deflection, kappa and gamma caused by a collection of halos with truncated Jaffe mass profiles. 

The profile is $ \rho \propto \left( \frac{r}{r_s} \right)^{-2} \left( 1 + \frac{r}{r_s} \right)^{-2} $ so beta would usually be negative.
