\doxysubsection{Detailed Description}
A class that includes the MOKA lens map. 

A class, where the lens is represented by a MOKA cluster in the form of a \doxylink{struct_m_o_k_amap}{MOKAmap} object (see the description of \doxylink{struct_m_o_k_amap}{MOKAmap}). It can either be used with the \doxylink{class_lens}{Lens} model or by itself.

Note\+: To use this class requires setting the ENABLE\+\_\+\+FITS compiler flag and linking the cfits library.
