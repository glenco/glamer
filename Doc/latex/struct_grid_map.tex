\doxysubsection{Detailed Description}
A simplified version of the \mbox{\hyperlink{struct_grid}{Grid}} structure for making non-\/adaptive maps of the lensing quantities (kappa, gamma, etc...) 

Grip\+Map is faster and uses less memory than \mbox{\hyperlink{struct_grid}{Grid}}. It does not construct the tree structures for the points and thus cannot be used for adaptive mapping or image finding.

The distance between the left (lower) most and right (upper) most ray is range so the resolution is range/(N-\/1). The lower left pixel is at center\mbox{[}\mbox{]}-\/0.\+5$\ast$range and the upper right is at center\mbox{[}\mbox{]}+0.\+5$\ast$range
