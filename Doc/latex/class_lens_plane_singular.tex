\doxysubsection{Detailed Description}
A \mbox{\hyperlink{class_lens_plane}{Lens\+Plane}} with a list of \mbox{\hyperlink{class_lens_halo}{Lens\+Halo}}\textquotesingle{}s in it. ~\newline
 

The deflection is calculated by direct summation which can be slow for large numbers of \mbox{\hyperlink{class_lens_halo}{Lens\+Halo}}\textquotesingle{}s. ~\newline
 Main lenses are put onto these planes when they are added to the \mbox{\hyperlink{class_lens}{Lens}}.
