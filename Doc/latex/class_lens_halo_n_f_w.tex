\doxysubsection{Detailed Description}
A class for calculating the deflection, kappa and gamma caused by a collection of N\+FW halos. 

This class uses the true expressions for the N\+FW profile. This is time consuming and not usually necessary. See Tree\+Quad\+Pseudo\+N\+FW for a faster alternative.

The default value of theta = 0.\+1 generally gives better than 1\% accuracy on alpha. The shear and kappa is always more accurate than the deflection.
