\doxysubsection{Detailed Description}
Structure for reading and writing parameters to and from a parameter file as well as a container for passing the parameters to other classes and functions. 

The constructor reads in all parameters in the parameter file. They can then be accessed with the get functions. There should be no interaction with the parameter file except through the \mbox{\hyperlink{class_input_params}{Input\+Params}} structure.
