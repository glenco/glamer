\doxysubsection{Detailed Description}
A class for calculating the deflection, kappa and gamma caused by a collection of halos with truncated power-\/law mass profiles. 

The truncation is in 2d not 3d. $ \Sigma \propto r^{-\beta} $ so beta would usually be positive.

The default value of theta = 0.\+1 generally gives better than 1\% accuracy on alpha. The shear and kappa is always more accurate than the deflection.
