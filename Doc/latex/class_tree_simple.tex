\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename PType$>$\newline
class Tree\+Simple$<$ PType $>$}
A C++ class wrapper for the bianary tree\+NB used in the Nbody force calculation, but also useful for general purpose searches. 

Most of the code in Tree\+NB.\+c and Tree\+Driver\+NB.\+c is duplicated here as private methods and a few public ones.

PType must have a dereferencing operator \mbox{[}\mbox{]} that gives the value to be sorterd. THIS CAN ONLY BE IN 2 OR 3 DIMENSIONS.
