\doxysubsection{Detailed Description}
Class for sources described by a Sersic profile. 

The sources are constructed from magnitude, half light radius Reff, Sersic index, axis ratio q and main axis orientation PA. For q = 1, the source is circular and the surface brightness profile is I(r) = Ieff $\ast$ exp(-\/bn$\ast$((r/\+Reff)$^\wedge$(1/index)-\/1)) The elliptical model is defined for axis ratio q $<$ 1.
