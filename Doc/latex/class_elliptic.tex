\doxysubsection{Detailed Description}
Class to convert isotropic Lens\+Halos into elliptical halos. 

This class changes the mass distribution into an elliptical shape using the approach of T. Schramm, 1990 A\&A, 231,19. The mass is conserved. An isotropic \doxylink{class_lens_halo}{Lens\+Halo} needs to be constructed first and then it is past to \doxylink{class_elliptic}{Elliptic}.
