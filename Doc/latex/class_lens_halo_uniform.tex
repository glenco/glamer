\doxysubsection{Detailed Description}
A uniform surface density and shear lens. 

Note the units of the input shear and surface density are not Sigma\+\_\+crit\textquotesingle{}s. This lens will extend to infinity in the angular directions.

If the magnitude g and position angle, theta, are specified then

Shear\mbox{[}0\mbox{]} = g $\ast$ cos( 2$\ast$theta ) Shear\mbox{[}1\mbox{]} = g $\ast$ sin(  2$\ast$theta )
