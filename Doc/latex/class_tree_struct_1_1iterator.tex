\doxysubsection{Detailed Description}
A iterator class fore \doxylink{struct_tree_struct}{Tree\+Struct} that allows for movement through the tree without changing anything in the tree itself. 

This class should be able to preform all of the constant movements within the tree without causing any change to the tree.
