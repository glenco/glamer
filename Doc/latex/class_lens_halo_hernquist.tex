\doxysubsection{Detailed Description}
A class for calculating the deflection, kappa and gamma caused by a collection of halos with truncated Hernquist mass profiles. 

The profile is $ \rho \propto \left( \frac{r}{r_s} \right)^{-1} \left( 1 + \frac{r}{r_s} \right)^{-3} $.
