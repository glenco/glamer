\doxysubsection{Detailed Description}
Class for calculating properties of N\+FW halo profile. 

This class does not take into affect the cosmological correlation between concentration and mass. For this see the \mbox{\hyperlink{class_h_a_l_o_calculator}{H\+A\+L\+O\+Calculator}} class.
